## 🖥️ What this app does

license-plate-recognition watches a camera feed and reads vehicle license plates in real time. It uses object detection to find the plate, then OCR to read the text. It can also save plate data to a database for later review.

This app is built for simple use on Windows. You do not need to know Python or install extra tools in most cases. Download the release, open the app, and start scanning plates

## ✨ Main features

- Real-time license plate detection
- Plate text reading with OCR
- Camera feed support
- Database logging for plate records
- Works with common Windows webcams and video sources
- Clear results that show the plate text on screen
- Built for traffic and security use cases

The app uses two steps:

1. It finds the license plate in the video frame

The app may also save each read to a database with a time stamp. This makes it easier to review plate activity later. The database log can help with gate access, site security, parking checks, and traffic review

## 🛠️ Common problems and fixes

### The app does not open

- Make sure the download finished fully
- Extract the ZIP file before opening the app
- Right-click the app and choose Run as administrator
- Check that your antivirus did not block the file

### The camera does not show video

- Close other apps that use the camera
- Unplug and reconnect the camera
- Restart the app
- Try a different USB port
- Check Windows camera privacy settings

### The plate text is wrong

- Move the camera closer
- Improve lighting
- Reduce glare
- Keep the camera still
- Make sure the plate is not blurred by motion

### The app feels slow

- Close other heavy apps
- Use a stronger PC if possible
- Lower the video size if the app has that setting
- Use a direct USB camera instead of a low-quality capture device

## 📌 Tips for better results

- Use bright, even light
- Keep the camera level with the plate
- Avoid sharp angles
- Use a clear view of the full plate
- Test with a parked car before live use
- Review the output before using it in a real workflow
